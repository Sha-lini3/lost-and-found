# 🔍 Lost & Found Portal

A modern **Lost & Found Portal** developed using **HTML, CSS and JavaScript** to simplify the process of reporting, searching and recovering lost items. The platform provides an easy-to-use interface for users to report lost or found belongings while allowing administrators to manage all submitted reports.

---

## ✨ Features

- 🏠 Modern Landing Page
- 👤 User Login & Signup
- 📋 Report Lost Items
- 📦 Report Found Items
- 🔎 Search Lost Items
- 📂 Category-wise Item Display
- 📊 Admin Dashboard
- ✏️ Edit & Delete Reports
- 💬 Feedback Form
- 📞 Contact Us Page
- 🎤 Voice Input Support (Speech Recognition)
- 📱 Fully Responsive UI
- 💾 LocalStorage Based Data Management

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API
- Web Speech API

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/Sha-lini3/lost-and-found.git
```

2. Open the project folder.

3. Open **Firstpage.html** in your browser or run it using **Live Server** in VS Code.
